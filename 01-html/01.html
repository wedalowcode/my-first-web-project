Hello World - 我的第一个HTML页面

学习创建一个简单的HTML页面，这是Web开发的第一步

Hello World

这是我的第一个网页！

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一个页面</title>
    <style>
        body {
            font-family: sans-serif;
            line-height: 1.6;
            margin: 2em;
            background-color: #f4f4f4;
        }
        .container {
            max-width: 800px;
            margin: auto;
            background: #fff;
            padding: 20px;
            border-radius: 8px;
            box-shadow: 0 0 10px rgba(0,0,0,0.1);
        }
        h1 {
            color: #333;
            text-align: center;
            border-bottom: 2px solid #eee;
            padding-bottom: 10px;
        }
        p {
            color: #555;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Hello World</h1>
        <p>这是我的第一个网页！</p>
    </div>
</body>
</html>

HTML基础结构解析

文档类型声明

<!DOCTYPE html> - 这是HTML5的文档类型声明，告诉浏览器这是一个HTML文档。

HTML元素

<html lang="zh-CN"> - 这是HTML文档的根元素，lang属性指定文档的语言为中文。

头部信息

<head> - 包含文档的元数据，如标题、字符集、视口设置和样式。

  • <meta charset="UTF-8"> - 设置文档字符编码为UTF-8，支持各种语言字符。
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> - 设置视口，使网页在移动设备上正确显示。
  • <title> - 设置网页标题，显示在浏览器标签上。
  • <style> - 包含CSS样式规则，用于美化网页。

页面主体

<body> - 包含网页的可见内容。

  • <div class="container"> - 一个容器元素，用于组织内容并应用样式。
  • <h1> - 一级标题，通常用于页面的主标题。
  • <p> - 段落元素，用于显示文本内容。

CSS样式解析

样式规则定义了页面的视觉外观：

  • 设置了字体、行高和背景色
  • 创建了一个居中的白色容器，带有圆角和阴影
  • 美化了标题和段落文本
